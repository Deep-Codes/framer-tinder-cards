### Built with

- Nextjs
- Typescript
- Framer motion

### Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

To run Playwright e2e Tests

```bash
yarn test
```
